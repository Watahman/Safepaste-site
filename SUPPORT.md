# Support

For installation, configuration, and Team-policy help, consult the documents
in [docs](../docs) first.

When asking for support, include your SafePaste version, browser version, and
sanitized steps to reproduce the problem. Never share API keys, tokens,
passwords, private keys, customer information, or unredacted screenshots.

Security vulnerabilities must be reported privately as described in
[SECURITY.md](../SECURITY.md).
